Project Showcase: Neni Cookies Home Made Industry Website

Project Description

This website serves as a landing page and product catalog for Neni Cookies Home Made Industry, a home-based business specializing in the production and sale of homemade cookies. The primary goal of this website is to establish a digital presence, enhance business credibility, and expand marketing reach to potential customers through a professional, informative, and user-friendly visual interface.

The site features several key sections designed with a clean and intuitive layout. It starts with a business introduction, a brief company profile, ordering information, and a complete catalog of the various cookies offered. Visitors can view product photos with prices and are seamlessly directed to WhatsApp for quick and easy ordering. The website also includes an FAQ (Frequently Asked Questions) section to address common customer inquiries.

Technologies Used

The development of the Neni Cookies website utilized a combination of modern and structured front-end technologies to deliver a responsive, lightweight, and user-friendly experience.

HTML5 & SCSS (Sass)

The index.html file forms the main framework of the website, built with a clean, semantic structure compliant with modern standards. SCSS was used as a CSS preprocessor, enabling a more modular, organized, and scalable stylesheet. The `scss/` folder is divided into sections like `base/`, `components/`, `layout/`, and `theme/` to separate design features such as colors, buttons, and grid layouts.

JavaScript (Vanilla JS)

Two main JavaScript files were implemented: main.js contains custom scripts for user interactions like scroll animations, and scrollreveal.min.js is an external library used for scroll-based animations, creating a dynamic and interactive feel as elements appear on the screen.

Optimization and Responsiveness

The website is designed to be fully responsive, ensuring a comfortable viewing experience on both desktop and mobile devices. Product images are compressed to ensure fast page loading times without sacrificing visual quality. Each product features a WhatsApp icon with a pre-configured link to the seller's number and an automated message, simplifying the ordering process to a single click.

Conclusion

The Neni Cookies Home Made Industry website is a prime example of how small and medium-sized enterprises (SMEs) can be digitized, focusing not only on an attractive visual appearance but also on technical aspects like code structure, access speed, and user experience.

By combining HTML, SCSS, and JavaScript with an efficient file structure, this website serves as a powerful promotional tool that directly helps increase sales. This project demonstrates my ability to build a website that is functional, efficient, and ready for real-world business needs.
